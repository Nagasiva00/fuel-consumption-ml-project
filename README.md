# Machine Learning Model for Predicting Average Fuel Consumption

This project predicts the average fuel consumption of heavy vehicles using a Machine Learning model based on Artificial Neural Networks (ANN).
A simple graphical user interface (GUI) is implemented using Python Tkinter to upload datasets, train the model, make predictions, and visualize results.

## Technologies Used

* Python
* Machine Learning
* Artificial Neural Network (ANN)
* Pandas
* NumPy
* Tkinter (GUI)
* Matplotlib

## Features

* Upload dataset in CSV format
* Train ANN model using vehicle parameters
* Predict average fuel consumption
* Visualize prediction results using graphs

## Project Workflow

1. Upload dataset
2. Train ANN model
3. Predict fuel consumption
4. Display results and graph visualization

## How to Run the Project

1. Install required libraries:

pip install pandas numpy scikit-learn tensorflow matplotlib

2. Run the project:

python fuel_model.py
